## Введение

Проектная практика была направлена на получение начальных практических навыков работы с современными информационными технологиями. В ходе практики были использованы Git, Markdown, HTML, CSS и Python.

Основная часть работы включала подготовку репозитория, оформление документации, создание статического сайта и размещение материалов в GitHub и GitVerse. Вариативная часть была посвящена практической реализации базового HTTP-сервера на Python.

## 1. Общая информация о проекте

### Название проекта

Создание базового HTTP-сервера на Python.

### Цели и задачи проекта

Цель проекта - изучить базовые принципы работы протокола HTTP и реализовать учебный веб-сервер без использования готовых веб-фреймворков.

Задачи проекта:

- изучить структуру HTTP-запроса и HTTP-ответа;
- разобраться с работой TCP-сокетов в Python;
- реализовать обработку GET-запросов;
- настроить отдачу HTML и CSS файлов из папки сайта;
- подготовить документацию и статический сайт;
- разместить результаты в GitHub и GitVerse.

## 3. Описание задания по проектной практике

В рамках базовой части практики необходимо было подготовить Git-репозиторий, оформить материалы в Markdown, создать статический сайт на HTML/CSS и подготовить отчет.

Для вариативной части была выбрана тема из репозитория `codecrafters-io/build-your-own-x`: **Python : Building a basic HTTP Server from scratch in Python**.

Практическая часть включала создание простого HTTP-сервера на Python. Сервер запускается на локальном адресе `127.0.0.1:8080`, принимает GET-запросы, ищет файлы в папке `site` и возвращает HTTP-ответ клиенту.
