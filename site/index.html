Базовый HTTP-сервер на Python

Учебный проект по созданию простого веб-сервера с нуля: от TCP-сокетов до отдачи HTML-страницы браузеру.

О проекте

Проект посвящен изучению принципов работы протокола HTTP и созданию минимального сервера на Python без использования веб-фреймворков.

Участники

Тарасенко Роман Олегович - выполнение практики, реализация HTTP-сервера, подготовка документации и сайта.

Журнал

Выбор темы

Для вариативной части была выбрана тема создания HTTP-сервера на Python.

Изучение HTTP

Были рассмотрены HTTP-запросы, ответы, статус-коды и заголовки.

Реализация сервера

Создан сервер, который обрабатывает GET-запросы и отдает файлы сайта.
