2.2 Практическая реализация технологии «с нуля»

Вариативная часть посвящена созданию базового HTTP-сервера на Python. Сервер реализован без Flask, Django и других веб-фреймворков.

Что реализовано

  • запуск TCP-сервера на 127.0.0.1:8080;
  • обработка GET-запросов;
  • отдача HTML и CSS файлов из папки site;
  • ответы 200 OK, 400 Bad Request и 404 Not Found;
  • определение типа содержимого по расширению файла.

Схема работы

Браузер -> GET-запрос -> src/main.py -> site/index.html -> HTTP-ответ

Файлы

  • src/main.py - исходный код сервера;
  • docs/variant_guide.md - техническое руководство;
  • site/ - статические файлы, которые отдаёт сервер.
