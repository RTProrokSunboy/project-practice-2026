## Вариативная часть

**Python : Building a basic HTTP Server from scratch in Python**

Тема выбрана из подборки [codecrafters-io/build-your-own-x](https://github.com/codecrafters-io/build-your-own-x), раздел **Build your own Web Server**.

Цель вариативной части - изучить базовые принципы работы HTTP и реализовать простой HTTP-сервер на Python.

## Структура репозитория

- `docs/` - документация по практике и вариативной части.
- `reports/` - материалы итогового отчета.
- `site/` - статический сайт о проекте.
- `src/` - исходный код HTTP-сервера.
- `task/` - материалы задания.

## Быстрый запуск

```bash
python src/server.py
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8080
```
