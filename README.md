## Что находится в репозитории

- `docs/` - Markdown-документация по практике.
- `site/` - статический сайт по практике.
- `src/` - исходный код вариативной части.
- `reports/` - итоговый отчёт по практике.
- `task/` - исходное задание и материалы практики.
