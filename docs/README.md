# Документация по проектной практике

В этой папке размещены Markdown-материалы по обязательной части практики и по вариативному заданию.

## Основные файлы

- `project.md` - общая информация о практике, цели и задачи.
- `members.md` - участник и личный вклад.
- `journal.md` - журнал выполнения практики.
- `resources.md` - полезные источники и ссылки.
- `variation.md` - описание вариативной части.
- `practice_documentation.md` - подробное руководство по реализации HTTP-сервера.

## Пока не заполнено
