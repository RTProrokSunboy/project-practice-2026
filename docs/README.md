# Документация по практике

В этом разделе собраны материалы по выполнению проектной практики.

## Основная тема

Практическая реализация технологии: создание базового HTTP-сервера на Python с нуля.

## Цель работы

Изучить основы работы протокола HTTP, принципы обработки клиентских запросов и формирования серверных ответов.

## Выполненные этапы

1. Выбрана тема вариативной части из `codecrafters-io/build-your-own-x`.
2. Подготовлена структура репозитория.
3. Реализован простой HTTP-сервер на Python.
4. Подготовлен статический сайт с описанием проекта.
5. Подготовлена документация для отчета.

## Результат

Реализован учебный HTTP-сервер, который принимает GET-запросы, формирует HTTP-ответы и отдает статические файлы из папки `site`.
