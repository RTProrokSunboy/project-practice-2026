# Проектная практика

## Общая информация

Проектная (учебная) практика выполняется индивидуально. Работа направлена на освоение базовых инструментов разработки и документирования: Git, Markdown, HTML, CSS и Python.

В рамках практики подготовлен репозиторий, оформлена документация, создан статический сайт и выполнена вариативная часть по реализации базового HTTP-сервера на Python.

## Цель

Получить практические навыки работы с репозиторием, документацией, веб-страницей и простым программным компонентом.

## Задачи

- настроить и вести Git-репозиторий;
- подготовить структуру проекта;
- оформить документацию в Markdown;
- создать статический сайт по практике;
- подготовить журнал выполнения работ;
- собрать полезные ресурсы;
- выполнить вариативную часть;
- подготовить итоговый отчёт.

## Результат

В результате выполнения практики подготовлены:

- репозиторий с материалами практики;
- сайт по практике;
- документация в папке `docs`;
- исходный код вариативной части в папке `src`;
- итоговый отчёт в папке `reports`.
